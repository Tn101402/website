OPEN LOCALLY
1. Extract the entire ZIP file.
2. Open the folder named "site".
3. Open "index.html" in Chrome, Edge, Safari, or Firefox.
4. Click Viva Box, AEON, or Glam Beauty to open each full case study.

WHAT IS INCLUDED
- A project-first home page: Viva Box / AEON / Glam Beauty.
- Three complete case-study pages.
- All supplied project artwork has been placed in the website.
- Motion: smooth project-page transitions, scroll reveals and image lightbox.
- Responsive layouts for desktop and mobile.

COLOUR NOTE
Some source artwork arrived in CMYK format, which browsers do not render consistently. Those images were exported as browser-safe sRGB copies for this website. No creative filters, recolouring, tinting, or image cropping have been applied to the supplied project artwork. The layout uses "contain" display to preserve every asset's original proportions.
